## Part A – Theory

### Q1. What is a function in JavaScript? Why do we use it?

**Answer:**
Function/method in a JavaScript is the block of code that is reusable for particular process.
We use it because it helps to reduce the multiple uses of that block of code again and again. It reduced the block of code.

---

### Q2. Difference between function declaration and function expression

**Answer:**  
function declaration is the declaration of the function at the start of the code to make sure it exist.
function expression is the expression of the function that is given after the execution of the block of code.

---

### Q3. What are parameters and return values?

**Answer:**  
parameters are the arguments or value that are called in the fuctions.
return values are the value or output that are given after the execution of particular function.

---

### Q4. What is an array? Why is it used?

**Answer:**  
array is a list of data with index of each data starting from 0.
it is used to store multiple data inside of a declared single variable.

---

### Q5. Difference between length and index in an array

**Answer:**  
length is the actual length or how many list are there in an array
index is the location of the data stored in the list in an array - starting from 0 index.

---

### Q6. Name and explain any three array methods

**Answer:**  
any three array methods are:
empty array = a[]
data array = a[1,2,3,4,5,10]
array inside array = a[{1,2,3,4},{1,2,5},{4,2,4}]
